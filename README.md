# Task Manager API

This is a simple Task Manager API built using FastAPI.

## Features
- Create a task
- Get all tasks
- Update a task
- Delete a task

## Technologies
- Python
- FastAPI

## How to run

1. Install dependencies:
pip install fastapi uvicorn

2. Run the server:
uvicorn main:app --reload

3. Open in browser:
http://127.0.0.1:8000/docs
